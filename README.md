# pylint-pytest

A Pylint plugin to suppress false positive pylint warnings about pytest fixtures.

## Installation

To install:

```bash
$ pip install pylint-pytest
```

## Usage

```bash
$ pylint --load-plugins pylint_pytest <path_to_your_sources>
```
